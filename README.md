# Heart Disease Prediction Using Decision Trees

## Overview
This project predicts whether a patient has heart disease using a Decision Tree Classifier.

## Dataset
Processed Cleveland Heart Disease Dataset.

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Accuracy
Cross-validation accuracy: 75.23%

## Run

pip install -r requirements.txt
python decision_tree.py
